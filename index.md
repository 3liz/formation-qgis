## Données

Dans le dossier `data`, vous pouvez avoir des données comme :
 
* IGN:
    * AdminExpress
    * OCS-GE si disponible
* Osm2Igeo
* Base comparateur de l'INSEE
* SRTM si besoin

Ces données sont gratuites et téléchargeables, selon une licence. [Lire ici.](./source_de_donnees.md)
